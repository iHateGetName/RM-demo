# RM-demo
实习期任务简要：
  基本实现所有的公共任务
玩法/操作说明：
  wasd控制移动，鼠标可以控制镜头向左向右转向，按空格键发射子弹，qe键控制发射角度高地，f键切换第一第三人称视角。
  先要攻击前哨战，摧毁前哨战之后，哨兵和基地无敌状态才会解除。摧毁哨兵后，对基地的攻击力会提升。摧毁基地游戏结束。
